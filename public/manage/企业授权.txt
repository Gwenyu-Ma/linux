1.添加企业管理员
/Enterprisemanager/enterpriseAdd
Method：POST
参数：
{
	'UserName' :'rising',  //用户名
	'PWD' : '123456', //密码
        'EMail' : 'rising@rising.com.cn',  //邮箱
        'PhoneNo' : '15910303731', //手机号
        'OName' : 'Rising',   //企业名称
        
}
返回值：
{
	"r":
	{
		"code":0,
		"action":0,
		"msg":"错误信息"
	}
}

4.中心管理员->企业管理页面统计数据
Enterprisemanager/getTotalNum
返回值：json
[authTotal]   瑞星总授权数
[authNum]	中心管理员已经授权的企业数
[notAuthNum]   中心管理员没有授权的企业数
[usersNum]	总企业数
[soonExpireNum]  即将过期的企业数

5.显示企业列表
Enterprisemanager/enterpriseList
参数：
{
	UserName  模糊用户名
	OName  企业名
	createStartTime  创建开始时间
	createEndTime  创建结束时间
	status 0未授权  1正常 2即将到期  3过期（失效）
}

6.企业列表点击子产品数

10.产品授权页面
/Authmanager/productAuth
返回值：
	如交互图左边授权信息
	{
		userNum   //授权用户数
		importNum  //导入授权次数
		authStartTime //授权开始日期
		authEndTime  //授权结束日期
		lastImportTime  //最后一次导入时间
		authStatus  //授权状态
		productNum //授权子产品数
	}
